# Executando localmente

## Pré requisitos
- Unity
- Visual Studio ou Visual Studio Code

## Execução
1. Faça o clone desse repositório
```
git clone https://github.com/fabricior0cha/unity_fps_game.git
```
2. Abra o `Unity Hub`
3. Clique em `Add > Add project from disk`
4. Selecione o local onde foi relizado o clone do projeto
5. Instale e importe os Assets utilizados
